# Artum Nakliyat Next.js Site

## Kurulum
1. GitHub'a yükle.
2. Vercel.com hesabına GitHub ile giriş yap.
3. New Project > bu repoyu seç > Deploy.
4. Vercel Domains bölümünden `artumnakliyat.com` ekle.
5. Squarespace DNS kayıtlarını Vercel'in verdiği değerlere göre güncelle.

## Geliştirme
npm install
npm run dev
